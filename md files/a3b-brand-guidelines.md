# a3b — Brand Guidelines

---

## Brand Name

**a3b** — lowercase, always. The `3` is the only element rendered in Sage (#7A8B6F). Never capitalised, never recoloured, never stretched.

---

## Brand Motive

> *Minimal videos. Calm delivery. Considered craft.*

a3b creates video content that respects the viewer's attention — clean cuts, unhurried pacing, no noise. The work speaks quietly and lands harder for it. Every edit is deliberate. Every frame earns its place.

**Voice:** short sentences. No exclamation points. Confident, not loud.
**Examples:** "Made for focused viewing." / "Less, but better." / "One story at a time."

## Brand Fonts

**Primary — Geist (Medium 500)**
Used for: wordmark, display headings, body copy, all UI text.
Download free: [vercel.com/font](https://vercel.com/font)

**Secondary — Geist Mono (Regular 400)**
Used for: timestamps, version labels, status tags, code paths. Never for body or headings.

**Scale:**
- Display: 48px / 500
- Heading: 32px / 500
- Sub: 20px / 500
- Body: 16px / 400
- Caption: 13px / 400
- Label: 11px / Mono

## Brand Logo

**Primary — Wordmark**
`a3b` in Geist Medium. The `3` is always Sage (#7A8B6F). All other letters in Ink (#1A1F1C) on light, or Bone (#F4F1EB) on dark.

**Secondary — Mark**
A sage rounded-square chip containing `3` in Geist Medium / Bone. Used for app icon, favicon, social avatar — anywhere the wordmark can't read below 60px wide.

**Variations:**
- Default: Ink wordmark on Bone background
- Dark mode: Bone wordmark on Ink background ← preferred
- Monochrome: All-Ink wordmark, no Sage (single-colour print only)

**Rules:**
- Clear space ≥ 1× the height of the `a` on all sides
- Minimum wordmark width: 60px — below that, use the mark
- Never stretch, rotate, recolour, or change the weight
- Never apply effects (shadow, glow, outline)
- The Sage colour appears **only** on the `3` — nowhere else in the logo
